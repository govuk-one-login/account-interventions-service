# Design docs

This directory is for storing design docs.
Design docs are used to communicate and align on a design for a new feature or component.
Design docs are **not** living documents. 
They should not be kept up to date after publishing, and should not be used to understand a feature or component that has been built.
Instead, we should use the code itself and associated documentation for that.
